# 《末日幸存者》快速上线指南

## 📊 项目概况

这是一个**极简版小游戏**，专门为快速赚钱而设计。

| 项目 | 详情 |
|------|------|
| 游戏类型 | 快速点击游戏 |
| 游戏时长 | 30秒/局 |
| 变现方式 | 广告变现（IAA） |
| 分发方式 | 微信分享 + 二维码 + 短链接 |
| 投资成本 | 0元 |
| 预期收入 | 月1000-5000元 |
| 上线时间 | 3-5天 |

## 🚀 快速开始

### 第一步：在本地运行游戏

```bash
# 克隆项目
git clone https://github.com/BloomingProsperity/doomsday-survivor.git
cd doomsday-survivor

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

然后访问 `http://localhost:3000/game-simple` 查看简化版游戏。

### 第二步：测试游戏

1. 打开游戏
2. 点击"开始游戏"
3. 在30秒内尽可能多地点击
4. 游戏结束后会显示分数
5. 点击"分享给朋友"测试分享功能

### 第三步：部署上线

#### 方案A：使用Manus平台（推荐）

1. 在Manus管理面板点击"Publish"
2. 选择"发布"
3. 获得公开链接
4. 分享链接给朋友

#### 方案B：使用其他平台

- **Vercel**（推荐）：https://vercel.com/
- **Netlify**：https://www.netlify.com/
- **GitHub Pages**：https://pages.github.com/

**使用Vercel部署步骤：**

```bash
# 1. 安装Vercel CLI
npm i -g vercel

# 2. 部署
vercel

# 3. 按照提示完成部署
# 4. 获得公开URL
```

## 💰 广告变现配置

### 支持的广告平台

| 平台 | 优点 | 缺点 | 收益 |
|------|------|------|------|
| Google AdSense | 全球通用 | 需要谷歌账号 | 高 |
| 腾讯广告 | 适合微信 | 需要企业资质 | 中 |
| 头条广告 | 适合抖音 | 需要头条账号 | 中 |
| 模拟广告 | 无需配置 | 无真实收益 | 测试用 |

### 配置广告

#### 1. 使用Google AdSense（最简单）

1. 访问 https://www.google.com/adsense
2. 注册账号
3. 添加网站
4. 获得广告代码
5. 在 `client/index.html` 中添加：

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-xxxxxxxxxxxxxxxx"
     crossorigin="anonymous"></script>
```

#### 2. 使用腾讯广告（适合微信）

1. 访问 https://ad.qq.com/
2. 注册账号
3. 创建广告位
4. 获得广告代码
5. 在游戏中集成

#### 3. 使用模拟广告（测试）

当前默认使用模拟广告，每3局游戏显示一次。

### 修改广告配置

在 `client/src/pages/GameSimplified.tsx` 中修改：

```typescript
const AD_CONFIG = {
  enabled: true,           // 是否启用广告
  showInterval: 3,         // 每3局显示一次
};
```

## 📱 分享功能配置

### 微信分享

游戏已内置微信分享功能。用户点击"分享给朋友"时会自动调用微信分享。

**需要配置：**

1. 在 `client/index.html` 中添加微信JS-SDK：

```html
<script src="https://res.wx.qq.com/open/js/jweixin-1.6.0.js"></script>
```

2. 配置微信权限（需要在微信公众平台设置）

### 二维码分享

点击"生成二维码"会自动生成当前页面的二维码。

### 短链接

游戏会自动生成短链接，方便分享。

## 🔧 常见问题

### Q1: 为什么没有收入？

**可能原因：**
1. 广告还未配置
2. 用户数量太少
3. 广告平台还未审核通过

**解决方案：**
1. 配置Google AdSense
2. 邀请更多朋友玩
3. 等待广告平台审核

### Q2: 如何修改游戏内容？

在 `client/src/pages/GameSimplified.tsx` 中修改：

```typescript
// 修改游戏时间（当前30秒）
setTimeLeft(30);  // 改成其他数字

// 修改游戏标题
<h1 className="text-4xl font-bold text-white mb-2">末日幸存者</h1>

// 修改分数倍数
setScore((prev) => prev + 1);  // 改成 prev + 2 可以加倍分数
```

### Q3: 如何添加新功能？

使用Cursor的AI功能：

1. 打开 `GameSimplified.tsx`
2. 选中代码
3. 按 `Ctrl+K`（Mac: `Cmd+K`）
4. 描述您的需求
5. AI会生成代码

### Q4: 游戏在手机上怎么玩？

1. 部署到公网
2. 在手机浏览器中打开链接
3. 或生成二维码扫描

### Q5: 如何追踪收入？

1. 在Google AdSense后台查看
2. 或在浏览器控制台查看本地统计数据

## 📈 收入预估

基于以下假设：

- 日活用户：100人
- 平均每人玩3局
- 每3局显示1次广告
- 每次广告收益：0.1元（模拟）

**月收入预估：**

```
100人 × 3局 × 30天 = 9000局
9000局 ÷ 3 = 3000次广告
3000次 × 0.1元 = 300元/月

实际收益可能是这个的2-10倍（取决于广告平台和用户质量）
```
